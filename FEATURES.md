# 🌟 FlowBoard Features & Guide

This document provides a detailed overview of all features available in FlowBoard and how to use them effectively.

## ⌨️ Shortcuts & Quick Actions

| Action | Shortcut / Method | Description |
| :--- | :--- | :--- |
| **Quick Add Task** | `Ctrl + Shift + F` | Opens the Quick Add modal on **any** webpage. |
| **Quick Add (Mouse)** | Right-Click > "Add to FlowBoard" | Adds the current page or selected text as a task. |
| **Extension Icon** | Click Icon | Opens the Quick Add modal. |
| **Close Modal** | `Esc` | Closes any open modal. |
| **Save Task** | `Enter` (in input) | Saves the task (if configured). |

---

## 📊 Core Views

### 1. Kanban Board
-   **Drag & Drop:** Move tasks between columns to update status.
-   **Collapse Columns:** Use the collapse icon in the header to hide tasks in a specific column.
-   **Sort:** Tasks are automatically sorted by order index.

### 2. List View
-   **Compact Table:** View all tasks in a structured table format.
-   **Tags:** Full tag names are displayed for clarity.
-   **Sorting:** Click column headers to sort by Title, Assignee, Priority, etc.

### 3. Calendar View
-   **Timeline:** Visualize task deadlines on a monthly calendar.
-   **Drag to Reschedule:** Drag tasks to different dates to update their due date.

---

## ⏱️ Time Tracking

-   **Start/Stop Timer:** Click the **Play** button on any task card to start tracking time. Click **Pause** to stop.
-   **Manual Entry:** In the Task Edit modal, manually enter "Actual Time" if you forgot to run the timer.
-   **Visual Indicator:** Tasks with active timers show a pulsing green indicator.

---

## 📈 Reports (Managers Only)

Access the **Reports** icon in the top bar to view:
-   **Total Time Tracked:** Aggregate time spent on the project.
-   **Task Completion:** Total vs. Completed tasks.
-   **User Performance:** Breakdown of tasks assigned and time tracked per user.

---

## 💾 Data Management

Access the **Data** icon (Database symbol) to:
-   **Export CSV:** Download a spreadsheet of all tasks, including time data and assignees.
-   **Export JSON:** Download a full raw data dump of the project.
-   **Archive Project:** Remove the project from your active workspace (Soft Delete).

---

## 👥 Team Collaboration

-   **Roles:**
    -   **Manager:** Full access (Settings, Reports, Delete Project).
    -   **Lead:** Can manage tasks, columns, and team, but cannot delete project.
    -   **Resource:** Can view and move tasks, but has limited management rights.
-   **Real-time Sync:** Changes made by one member appear instantly for others.

---

## 🎨 Customization

-   **Dark Mode:** Toggle between Light and Dark themes in the side menu.
-   **Tags:** Create custom tags with colors for Priority, Type, or any other category.
-   **Project Theme:** Set a custom primary color for each project.
