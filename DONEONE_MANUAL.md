## <a id="introduction"></a>1. Introduction

**DoneOne** is a premium, high-performance task management extension designed for professionals who need to manage projects without leaving their browser. It combines the visual utility of a Kanban board with the speed of keyboard-centric workflows, all wrapped in a modern, aesthetic interface.

### Key Benefits
*   **Seamless Integration:** Capture tasks from any webpage instantly.
*   **Visual Clarity:** Switch between Kanban, List, and Calendar views.
*   **Data Control:** Full ownership of your data with export capabilities.
*   **Team Sync:** Real-time collaboration updates.

## <a id="getting-started"></a>2. Getting Started

### Installation
1.  **Load the Extension:** In Chrome, go to `chrome://extensions/`, enable Developer Mode, and load the unpacked extension.
2.  **Pin it:** Pin the DoneOne icon to your browser toolbar for quick access.
3.  **Account Setup:** Click the icon and sign up to create your profile.

### Your First Project
1.  **Create:** On the dashboard, click the **+** button to create a new project.
2.  **Invite:** Share the unique **Join Code** (found in Project Settings) with team members.
3.  **Manage:** Use the default columns (Pending, In Progress, Done) or customize them to fit your workflow.

## <a id="core-workflows"></a>3. Core Workflows

### ⚡ Quick Add Task
Capture ideas instantly without context switching.
*   **Shortcut:** Press `Ctrl + Shift + F` on any webpage.
*   **Popup:** A modal appears asking for Task Title, Project, and Assignee.
*   **Smart Defaults:** The project defaults to your last used (or first available), and Assignee defaults to "Me".
*   **Context Menu:** Right-click any text or page background and select **"Add to DoneOne"**.
*   **Extension Icon:** Click the icon in the toolbar.

### Managing Tasks
*   **Create:** Click "Add Task" at the bottom of any column.
*   **Edit:** Click a task card to open the **Task Edit Modal**. Here you can set:
    *   **Assignee:** Who is responsible.
    *   **Priority:** Low, Medium, High.
    *   **Due Date:** Set deadlines.
    *   **Description:** Rich text details.
*   **Move:** Drag and drop tasks between columns to update their status.
    *   *Auto-Move:* Moving to "In Progress" sets a start time. Moving to "Done" sets a completion time.

### ⏱️ Time Tracking
*   **Start Timer:** Click the **Play (▶)** button on a task card.
*   **Stop Timer:** Click **Pause (||)** to log the session.
*   **Manual Entry:** Forgot to track? Open the task detail and enter "Actual Time" manually.

## <a id="view-modes"></a>4. View Modes

Switch views using the toggle in the top-right corner of the board.

### 📋 Kanban Board (Default)
The classic visual workflow.
*   **Best for:** Visualizing process flow and bottlenecks.
*   **Features:** Drag-and-drop, column collapsing, visual tags.

### 📝 List View
A compact, data-dense table.
*   **Best for:** Processing large backlogs or sorting by priority/assignee.
*   **Features:** Sortable headers, compact rows, quick status updates.

### 📅 Calendar View
A timeline perspective.
*   **Best for:** Planning deadlines and scheduling.
*   **Features:** Drag tasks to reschedule, monthly overview.

## <a id="advanced-features"></a>5. Advanced Features

### 🔍 Search & Filters
*   **Global Search:** Use the search bar to find tasks by title or content.
*   **Filtering:** Click the **Filter** icon to view tasks by:
    *   Assignee
    *   Priority
    *   Due Date

### 📂 History & Archives
Access the **History** tab to see completed and archived tasks.
*   **Retention:** Old history is automatically archived based on Admin policy.

## <a id="administration--teams"></a>6. Administration & Teams

### 👥 User Roles
*   **Manager:** The project owner. Has full control, including deleting the project and managing all settings.
*   **Lead:** Can manage tasks, columns, and team members, but cannot delete the project.
*   **Resource:** Can view and move tasks. Limited administrative access.

### 📊 Reports (Managers Only)
Access the **Reports** dashboard for insights:
*   **Status Distribution:** Pie charts of task progress.
*   **Workload Analysis:** See which team members are overloaded.
*   **Time Reports:** Aggregate hours tracked per user or project.

### ⚙️ Admin Panel
Located in the sidebar, this area allows global configuration:
*   **User Management:** Approve/Disable users.
*   **Image Uploads:** Toggle upload permissions globally or per-user.
*   **Registration:** Close registration to prevent new sign-ups.

### 💾 Data Export
*   **CSV Export:** Download a spreadsheet compatible with Excel/Sheets.
*   **JSON Export:** Full data dump for backups.

## <a id="troubleshooting"></a>7. Troubleshooting

| Issue | Solution |
| :--- | :--- |
| **Extension not opening** | Ensure Developer Mode is on and reload the extension. |
| **Sync issues** | Check your internet connection. The "Cloud" icon should be green. |
| **Cannot upload images** | Contact your Manager to enable upload permissions for your account. |
| **Forgot Password** | Contact the system administrator to reset credentials. |
